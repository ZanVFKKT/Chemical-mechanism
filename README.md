# Chemical-mechanism

The project includes Python programs applying a chemical mechanism to explain oscillations in absorbance in the H₂O₂–AgNPs system observed by LSPR.

## Modules

- **Parameter_generator.py**  
  Identifies suitable dimensionless parameters that satisfy the conditions for a single bistable AgNP system.

- **Phase_diagram.py**  
  Plots a phase diagram of a single bistable AgNP system for given dimensionless parameters.

- **Phase_diagram_coupled.py**  
  Plots a phase diagram of two coupled bistable AgNPs with a given interaction strength and initial conditions.

- **Time_evolution_of_theta.py**  
  Plots surface coverage as a function of time.
