# Chemical-mechanism

The project includes Python programs applying a chemical mechanism to explain oscillations in absorbance in the H₂O₂–AgNPs system observed by LSPR.

## Modules

- **Phase_diagram.py**  
  Plots a phase diagram of a single bistable AgNP system for given dimensionless parameters.

- **Phase_diagram_coupled.py**  
  Plots a phase diagram of two coupled bistable AgNPs with a given interaction strength and initial conditions.

- **Time_evolution_of_theta.py**  
  Plots surface coverage as a function of time.

- **Iskanje I za h.py**
  Determines the interaction strength (I) as a function of the coupling parameter (h) for the coupled bistable AgNP system.

- **iskanje I.py**
  Identifies suitable values of the interaction strength (I) that satisfy the required conditions for the coupled bistable AgNP system.

- **vpliv h na periodo.py**
  Analyzes the influence of the coupling parameter (h) on the oscillation period of the coupled AgNP system.

- **data.xlsx**
  Contains the numerical and experimental data used as input or generated by the simulations and analysis scripts.
